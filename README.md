# MyNewsPartner

## Basic Description 
MyNewsPartner is a news application that uses News API to fetch the data. Users can get an update of news according to the categories for eg. Sports,Business,Politics,etc.

## Tools
- News API
- HTTP Networking
- Retrofit 
- Picasso library
